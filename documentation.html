display property

    4 types of display are present for any block element
  • block:-reserves all the width of screen in left and right direction of element
  • inline:-reserves only space equal to content + padding border margin we cannot change the dimension of block
  • inline-block:-it has same property as inline but in addition we can change dimension of block
  • none:-this property makes block as if it is not present there
    • here is another tag which has similar properties as none
    • visibility tag:-this makes the block invoisible but reserves the space of block as it suppused to be(use to hide elements)
    • here is another tag span used to contain anything in inline box

positioning

    to position different inline block boxes on page
    there are 4 types of positioning
  1. relative:-it is used to position box relative to the position given by html by default.there are 4 types of positioning
    1. left
    2. top
    3. right
    4. bottom
    it does not effect the position of other blocks rather if position of two blocks coincides then one of the block get overlaped
  2. absolute
    3. as the name suggests its just straight forward positioning
  3. fixed:-it is special as it keeps the box as it is on the same position relative to your display sreen
